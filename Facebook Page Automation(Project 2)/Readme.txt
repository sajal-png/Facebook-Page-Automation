How To Run:
1. Install Eclipse IDE for Java Development
2. Create a maven project and select an archetype
3. You can choose org.apache.maven.archetypes and select quicstart groupid
4. Create a java class in src/main folder and run the code
5. Before running, we need a google alert setup on the email id, download chrome driver and specify its path in the code.
6. Specify facebook page/group link in the code.
7. Execute the Code.
